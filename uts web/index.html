Pengertian & perbedaan antara Client Side Scripting dengan Server Side Scripting

Client-side scripting adalah kode yang dieksekusi di sisi pengguna, biasanya di dalam browser web. Script ini berjalan pada perangkat pengguna setelah halaman web diunduh dari server. Contoh Client Side Scripting yaitu HTML, CSS, JAVASCRIPT, Animasi menu dropdown, Validasi formulir

Macam-macam jenis protokol dan contohnya

Server-side scripting adalah kode yang dieksekusi di sisi server. Script ini berjalan di server sebelum halaman web dikirim ke browser pengguna. Contoh Server Side Scripting yaitu PHP, Pyton, Ruby, Node.JS, Login pengguna dan autentikasi, Toko online

Pengertian Web Server

Web server adalah perangkat lunak yang dijalankan di komputer server yang menyimpan, memproses, dan mengirimkan halaman web kepada klien (web browser) melalui protokol HTTP atau HTTPS.

Pengertian Web Browser

Web browser adalah perangkat lunak aplikasi yang dijalankan di komputer atau perangkat mobile pengguna, yang digunakan untuk mengakses, menampilkan, dan berinteraksi dengan konten web. Contohnya Google Chrome, Mozilla, Microsoft Edge
